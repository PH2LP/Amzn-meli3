# Archivos Activos del Proyecto

## Script Principal
- **main2.py** - Pipeline completo Amazon → MercadoLibre CBT

## Módulos Core (src/)
- **category_matcher_v2.py** - Detección híbrida de categorías (embeddings + AI)
- **transform_mapper_new.py** - Transformación y mapeo de productos
- **mainglobal.py** - Publicación multi-país en MercadoLibre
- **amazon_api.py** - Descarga de productos desde Amazon SP-API
- **unified_transformer.py** - Transformación unificada
- **ai_validators.py** - Validación con IA
- **smart_categorizer.py** - Categorización inteligente

## Utilidades (utils/)
- **auto_refresh_token.py** - Renovación automática token MercadoLibre
- **auto_refresh_token_amzn.py** - Renovación automática token Amazon

## Datos Esenciales
- **resources/asins.txt** - Lista de ASINs a procesar
- **resources/data/cbt_categories_meta.json** - Metadata de categorías (513KB)
- **storage/category_cache/** - Embeddings y base de datos de categorías (17MB)
