## ğŸ“ Notas Importantes

1. **LaunchAgent vs Background**
   - LaunchAgent: Persiste entre reinicios del sistema (solo Token ML)
   - Background (nohup): Se detiene al reiniciar, debes reactivar manualmente

2. **Consumo de Recursos**
   - Todos los loops son muy livianos (~1-5 MB RAM cada uno)
   - No afectan el rendimiento del sistema
   - Los logs crecen ~1-10 MB por dÃ­a segÃºn actividad

3. **LÃ­mites de API**
   - MercadoLibre: 10,000 llamadas/dÃ­a (suficiente)
   - Amazon: 10 llamadas/segundo (suficiente)
   - OpenAI: SegÃºn tu plan (auto-respuestas optimizadas para minimizar uso)

4. **Persistencia**
   - Los servicios de background NO persisten entre reinicios
   - Debes crear scripts de inicio automÃ¡tico o usar cron/systemd
   - Para 24/7, migrar a VPS (ver [80_MIGRACION_VPS.md](80_MIGRACION_VPS.md))
